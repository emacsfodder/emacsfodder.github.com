    Purple Haze Theme

    Not my favorite Jimi Hendrix track, (that’d be All Along the Watchtower) but isn’t purple a nice restful color?

    This is yet another Emacs24 theme, it has core support for font-lock and some basic editor elements, but it lacks special support for Org, Helm, Gnus, Ido etc, etc.

    Grab it from github at http://jasonm23/emacs-purple-haze-theme
